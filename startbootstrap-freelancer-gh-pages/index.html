About

PeopleStream is a social media and news aggregator that provides users with information about thought leaders. It uses traditional media sources while helping readers navigate the problematic waters of social news by using screened sources to provide different perspectives on everyday topics.

Using NLP, it cautions users in order to address the disturbing reality in which the current state of media exists. With a goal to provide "complete news and promote media literacy, we're here to change the way people consume information."
